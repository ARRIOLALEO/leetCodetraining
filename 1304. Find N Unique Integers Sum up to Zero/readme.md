Given an integer n, return any array containing n unique integers such that they add up to 0.

 

### Example 1:

<pre>Input: n = 5
Output: [-7,-1,1,3,4]
Explanation: These arrays also are accepted [-5,-1,1,2,3] , [-3,-1,2,-2,4].</pre>
### Example 2:

<pre>Input: n = 3
Output: [-1,0,1]</pre>
### Example 3:

<pre>Input: n = 1
Output: [0]</pre>
 

### Constraints:

1 <= n <= 1000
